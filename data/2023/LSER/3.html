FORMATTAZIONE TESTO:

  1. B = bold (grassetto)

  2. U = underline (sottolineato)

  3. I = italic (corsivo)

  4. HR = horizontal line (linea orizzontale)

  5. BR = at the head (a capo)

  6. UL = lista NON ordinaria

  7. OL = lista ordinaria

  8. LI = lista item

  9. H1…H6 = grandezza carattere (titolo)

  10. a href= creare un collegamento ipertestuale a un'altra pagina web

  11. img SRC = inserire un'immagine nel documento

  12. table = creare una tabella contenente righe e colonne di dati

  13. tr = definisce una riga nella tabella.

  14. td = definisce una cella di dati all'interno di una riga della tabella

  15. form = crea un modulo per l'input degli utenti, come campi di testo, pulsanti e menu a tendina.

  16. input = crea un campo di input, come un campo di testo, casella di controllo o un pulsante di opzione.

  17. button = crea un pulsante interattivo

  18. textarea = crea un campo di testo multiriga per l'input dell'utente.
