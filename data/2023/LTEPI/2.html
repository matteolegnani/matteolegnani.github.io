1. Cd → Modifica la directory corrente

 

La Sintassi è:

cd [/d] [<drive>:][<path>]

  • cd .. → Passa alla directory superiore
  • cd [path]  → Passa al percorso specificato come argomento Il percorso può essere:  Relativo → a partire dalla directory corrente. Assoluto → a partire dalla radice del volume 
  • cd [drive:] → Passa alla radice del volume specificato 
  • cd\  →  Passa alla radice del volume corrente 
  • cd /?  → Mostra la guida e le possibili opzioni del comando

 

6. & → Concatena due  comandi

Esempio: dir & cd .. & dir

 

7. Mkdir → Crea una directory o una sottodirectory

La Sintassi e:

mkdir [<drive>:] <path>

  • mkdir [<drive>:]<path>→ Crea una directory nella posizione specificata 

 

 

 

8. Rmdir → Elimina una directory

La Sintassi è:

rmdir [<drive>:]<path> [/s [/q]]

  • rmdir [<drive>:] <path>→ Rimuove la directory specificata
  • rmdir [<drive>:] <path> /s →Rimuove l’albero a partire dalla directory specificata, chiede conferma dell’operazione tramite prompt all’utente 
  • rmdir [<drive>:] <path> /s /q →Specifica la modalità non interattiva. Non richiede conferma durante l'eliminazione di un albero di directory. Il parametro /q funziona solo se è specificato /s

 

9. copy → Copia di uno o più file da una posizione a un’altra
