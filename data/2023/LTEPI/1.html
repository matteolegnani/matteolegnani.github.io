PROMPT COMANDI

Comandi:

  1. Cls → cancella tutto il contenuto visualizzato nella finestra del Prompt dei comandi
  2. Exit → chiude la finestra del prompt dei comandi
  3. Color[XY] → permette di cambiare il colore dello sfondo e del testo (color senza parametri → reset)
  4. Dir → visualizza un elenco di file e sottocartelle della directory. Se usato senza parametri, visualizza l’etichetta del volume e il numero di serie seguito da un elenco di directory e file

     

La Sintassi è:

dir [<drive>:][<path>][<filename>][..]

  • dir /p: Visualizza una schermata (pagina) dell'elenco alla volta. Per visualizzare la schermata successiva, premere qualsiasi tasto. 
  • dir /q: Visualizza un elenco di file e cartelle con i relativi proprietari.
  • dir / W: Questa opzione viene utilizzata per visualizzare l'elenco dei file e delle directory in modo più conciso, senza fornire informazioni dettagliate come le dimensioni dei file, la data di creazione e l'ora di modifica.
  • dir /a [[:]<attributes>]
  • dir /o [[:]<sottorder>]
  • dir /?:Mostra la guida e le possibili opzioni del comando
