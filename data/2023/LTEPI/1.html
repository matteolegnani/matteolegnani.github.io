Sintassi

Comando (opzioni)(argomenti)

 

COMANDI = azione da eseguire

OPZIONI = chiamate switch o flag

ARGOMENTI = cartelle

 

Vi sì può accedere alla prompt dei comandi anche dalle cartelle

  • CLS → pulisce il terminale cioè pulisce tutta la shell
  • exit → chiude il terminale
  • Color → cambiare colore del testo o dello sfondo

 

I Comandi più importanti:

  • DIR → visualizza un elenco di file e sottocartelle della directory

 

visualizza anche 1 etichetta del volume, ci dice data e ora delle modifiche dei file

dir [<drive»>:][<path>]= PERCORSO RELATIVO

 

PERCORSO ASSOLUTO= riscrivo C:\Users\

 

Fleg

  1. dir/p: visualizza un elenco alla volta
  2. dir/q: visualizza i proprietari
  3. dir/W:
  4. dir/a[[:]<attributi>]

attributi:

Directory(d)(-d ti prende solo file)

File nascosti(h)(-h tutti i file non nascosti)

File sistema(s) ]

File solo lettura(r)

  1. dir/o[[:]<sottorder>]

    - n=alfabeticamente in base al nome

    - s=crescente per dimensione

  2. dir/?

Comando

  • cd → (Change Directory) Modifica la directory corrente, permette di modificare la posizione di lavoro

  • cd.. → passa alla directory genitore cioè quella superiore

  • cd [drive:] → permette di cambiare disco

  • cd\ → permette di ritornare alla radice del filesystem

  • cd /? → help guida alle possibili opzioni del comando

& = operatore che permette di concatenare 2 comandi (es: dir&cd..&dir)
