CICLO MACCHINA
Il ciclo macchina rappresenta le fasi fondamentali attraverso le quali un microprocessore esegue un'istruzione

  1. Fetch dell'istruzione: al centro della fase di fetch c'è un componente della CPU → Program Counter (PC), Il PC tiene traccia dell'indirizzo della prossima istruzione da eseguire
    - la CPU consulta il PC per sapere dove trovare l'istruzione
    - la CPU accede alla memoria e preleva l'istruzione 
    - l'istruzione viene inviata a una parte della CPU chiamata "Instruction Register" (IR) 
    - decodifica.
  2. Decode dell'istruzione: Decodificare un'istruzione → interpretare il significato
    → istruzioni memorizzate come sequenze di bit
    → la CPU deve tradurre queste sequenze in azioni concrete da eseguire →  decodifica
    - dopo che l’istruzione è stata prelevata dalla memoria e caricata nell’IR)
    - la CPU inizia la decodifica 
    - la CPU esamina l'istruzione per determinare quale operazione eseguire 
  3. Fetch degli operandi: se l'istruzione chiede dati aggiuntivi, la CPU li preleva dalla memoria 
    Dopo aver decodificato, la CPU ha bisogno di dati aggiuntivi per completare l'operazione. Questi dati, → operandi, sono essenziali, la fase riguarda proprio il prelevamento di questi dati dalla memoria
    quando l'istruzione è stata decodificata, la CPU sa quali operandi sono necessari, sono situati in:
    Registri interni: La CPU ha una serie di piccole unità di memoria chiamate registri
    Memoria principale: la CPU deve prelevarli dalla memoria principale
    - la CPU acceda alla posizione specificata e prelevi i dati necessari
    - ottenuti gli operandi, la CPU è pronta per l'esecuzione dell'istruzione
  4. Execute dell'istruzione: le istruzioni vengono trasformate in azioni
    - la CPU esegue operazioni aritmetiche, trasferisce dati da una posizione all'altra, prende decisioni 

    - ALU (Arithmetic Logic Unit): parte della CPU responsabile dell'esecuzione di operazioni-logiche
    - Registri: piccole unità di memoria all'interno della CPU, vengono utilizzate per conservare i dati durante l'esecuzione

RUOLO DEL CLOCK NEL CICLO MACCHINA
ogni fase è sincronizzata da un segnale di clock, esso determina quando la CPU dovrebbe passare alla fase successiva del ciclo. La frequenza del clock, indica quante volte al secondo il ciclo può essere completato
